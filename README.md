# Books List App (MVVM Pattern)

Cross platform ToDO app using flutter and dart with Hive Local Database 
 
# Run Locally

Clone the project

```bash
  git clone https://github.com/isnik09/ToDo-App.git
```

Go to the project directory

```bash
  cd ToDo-App
```

Install dependencies

```bash
  flutter pub get
```

Start in emulator

```bash
  flutter run
```
